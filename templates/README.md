# Online Library Website

This project is an Online Library Website that allows users to search for books, view details, and borrow books. The system also includes an admin role that can manage the books in the library.

## Pages Description

### Home.html

The home page of the website. It provides a general introduction to the online library and allows users to navigate to other pages such as login, signup, and book searching.

### Login.html

This page allows existing users or admins to log in to the system by entering their username and password.

### Signup.html

This page allows new users or admins to create an account by filling out the required information such as username, email, password, and selecting the account type.

### Navbar.html

Contains the navigation bar used across the website. It provides quick access to the main pages such as Home, Search Books, My Books, and other related pages.

---

## Admin Pages

### AdminAddBook.html

This page allows the admin to add a new book to the library by entering the book information such as book ID, name, author, category, and description.

### AdminEditBook.html

This page allows the admin to edit the details of an existing book in the library.

### MangeBook.html

This page displays the list of all books in the library and allows the admin to manage them, including editing or deleting books.

---

## User Pages

### SearchBook.html

Allows users to search for books in the library using different filters such as title, author, or category.

### BookDetails.html

Displays the detailed information about a selected book including its title, author, category, and description.

### BorrowBooks.html

Allows the user to borrow a book if it is available in the library.

### MyBooks.html

Shows the list of books that the current user has borrowed.
